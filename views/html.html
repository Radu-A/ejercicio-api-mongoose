Products / Providers - API

Endpoints para products

[GET] http://localhost:3000/api/products Retorna un objeto con los datos de todos los productos.

[POST] http://localhost:3000/api/products Para crear un nuevo producto

Endpoints para providers

[GET] http://localhost:3000/api/providers Retorna un objeto con los datos de todos los providers.

[POST] http://localhost:3000/api/providers Crea un nuevo proveedor
